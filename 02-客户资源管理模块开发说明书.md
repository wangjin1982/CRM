# 模块二：客户资源管理模块开发说明书

## 📋 模块概述

### 模块名称
客户资源管理模块（Customer Resource Management）

### 模块定位
CRM系统的核心数据模块，负责客户信息和联系人信息的管理，是整个系统的数据基础。

### 开发优先级
**P1（高优先级）** - 依赖基础设施模块，被商机、行为等模块依赖

---

## 🎯 模块目标

### 主要目标
1. 实现完整的客户信息管理（CRUD）
2. 实现客户名下多联系人管理
3. 实现客户信息的分类、标签、搜索
4. 实现客户360度视图（整合所有关联信息）
5. 实现客户导入导出功能

### 业务价值
- 建立企业统一客户档案
- 支持销售快速查找客户信息
- 为后续商机、行为模块提供数据基础

## 📐 数据库设计

### 表结构设计

#### 1. 客户信息表 (crm_customer_info)
```sql
CREATE TABLE crm_customer_info (
    id                  BIGSERIAL PRIMARY KEY,
    customer_no         VARCHAR(50) NOT NULL UNIQUE,  -- 客户编号
    customer_name       VARCHAR(200) NOT NULL,         -- 客户名称
    customer_type       VARCHAR(20) NOT NULL,          -- 客户类型: enterprise/individual

    -- 企业客户信息
    industry            VARCHAR(100),                  -- 所属行业
    company_size        VARCHAR(50),                   -- 公司规模
    legal_person        VARCHAR(50),                   -- 法人代表
    registered_capital  DECIMAL(20,2),                 -- 注册资本
    establish_date      DATE,                          -- 成立日期

    -- 联系信息
    province            VARCHAR(50),                   -- 省份
    city                VARCHAR(50),                   -- 城市
    district            VARCHAR(50),                   -- 区县
    address             VARCHAR(500),                  -- 详细地址
    website             VARCHAR(200),                  -- 官方网站

    -- 业务信息
    source              VARCHAR(50),                   -- 客户来源
    level               VARCHAR(20),                   -- 客户级别: A/B/C/D
    status              VARCHAR(20) DEFAULT 'active',  -- 状态: active/inactive/pool

    -- 负责人信息
    owner_id            BIGINT REFERENCES sys_user(id),-- 负责人ID
    owner_name          VARCHAR(50),                   -- 负责人姓名

    -- 统计信息
    opportunity_count   INT DEFAULT 0,                 -- 商机数量
    visit_count         INT DEFAULT 0,                 -- 拜访次数
    last_visit_at       TIMESTAMP,                     -- 最后拜访时间
    last_activity_at    TIMESTAMP,                     -- 最后活动时间

    -- AI增强字段
    ai_summary          TEXT,                          -- AI生成的客户画像摘要
    ai_insights         JSONB,                         -- AI分析洞察
    data_completed_at   TIMESTAMP,                     -- AI数据补全时间

    -- 元数据
    tags                VARCHAR(500)[],                -- 标签数组
    remarks             TEXT,                          -- 备注
    created_by          BIGINT REFERENCES sys_user(id),
    updated_by          BIGINT REFERENCES sys_user(id),
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    deleted_at          TIMESTAMP
);

-- 索引
CREATE INDEX idx_crm_customer_owner ON crm_customer_info(owner_id);
CREATE INDEX idx_crm_customer_type ON crm_customer_info(customer_type);
CREATE INDEX idx_crm_customer_level ON crm_customer_info(level);
CREATE INDEX idx_crm_customer_status ON crm_customer_info(status);
CREATE INDEX idx_crm_customer_tags ON crm_customer_info USING GIN(tags);
CREATE INDEX idx_crm_customer_name ON crm_customer_info(customer_name);
```

#### 2. 联系人信息表 (crm_contact_info)
```sql
CREATE TABLE crm_contact_info (
    id                  BIGSERIAL PRIMARY KEY,
    contact_no          VARCHAR(50) NOT NULL UNIQUE,   -- 联系人编号
    customer_id         BIGINT NOT NULL REFERENCES crm_customer_info(id), -- 所属客户

    -- 基本信息
    name                VARCHAR(50) NOT NULL,           -- 姓名
    title               VARCHAR(100),                  -- 职位
    department          VARCHAR(100),                  -- 部门
    gender              VARCHAR(10),                   -- 性别

    -- 联系方式
    mobile              VARCHAR(20),                   -- 手机
    phone               VARCHAR(20),                   -- 固话
    email               VARCHAR(100),                  -- 邮箱
    wechat              VARCHAR(50),                   -- 微信

    -- 决策链信息
    is_decision_maker   BOOLEAN DEFAULT FALSE,         -- 是否决策人
    is_influencer       BOOLEAN DEFAULT FALSE,         -- 是否影响者
    influence_level     SMALLINT,                      -- 影响力等级 1-5

    -- 关系信息
    relationship        VARCHAR(50),                   -- 关系类型
    preference          TEXT,                          -- 沟通偏好
    birthday            DATE,                          -- 生日
    hobbies             TEXT,                          -- 兴趣爱好

    -- 社交信息
    linkedin            VARCHAR(200),                  -- LinkedIn
    weibo               VARCHAR(200),                  -- 微博

    -- 状态
    status              VARCHAR(20) DEFAULT 'active',  -- 状态: active/inactive
    is_primary          BOOLEAN DEFAULT FALSE,         -- 是否主要联系人

    -- 统计信息
    contact_count       INT DEFAULT 0,                 -- 联系次数
    last_contact_at     TIMESTAMP,                     -- 最后联系时间

    -- 元数据
    remarks             TEXT,                          -- 备注
    created_by          BIGINT REFERENCES sys_user(id),
    updated_by          BIGINT REFERENCES sys_user(id),
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    deleted_at          TIMESTAMP
);

-- 索引
CREATE INDEX idx_crm_contact_customer ON crm_contact_info(customer_id);
CREATE INDEX idx_crm_contact_mobile ON crm_contact_info(mobile);
CREATE INDEX idx_crm_contact_email ON crm_contact_info(email);
CREATE INDEX idx_crm_contact_name ON crm_contact_info(name);
```

#### 3. 客户标签表 (crm_customer_tag)
```sql
CREATE TABLE crm_customer_tag (
    id          BIGSERIAL PRIMARY KEY,
    tag_name    VARCHAR(50) NOT NULL UNIQUE,
    tag_color   VARCHAR(20),
    tag_type    VARCHAR(20), -- system, custom
    sort_order  INT DEFAULT 0,
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 4. 客户交互记录表 (crm_customer_interaction)
```sql
CREATE TABLE crm_customer_interaction (
    id              BIGSERIAL PRIMARY KEY,
    customer_id     BIGINT NOT NULL REFERENCES crm_customer_info(id),
    contact_id      BIGINT REFERENCES crm_contact_info(id),

    interaction_type VARCHAR(50) NOT NULL, -- call, email, visit, wechat, other
    direction       VARCHAR(20),           -- inbound, outbound

    subject         VARCHAR(200),          -- 主题
    content         TEXT,                  -- 内容
    duration        INT,                   -- 时长(秒)

    attachments     VARCHAR(500)[],        -- 附件URL列表

    next_follow_at  TIMESTAMP,             -- 下次跟进时间
    next_follow_note TEXT,                 -- 下次跟进备注

    created_by      BIGINT REFERENCES sys_user(id),
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_crm_interaction_customer ON crm_customer_interaction(customer_id);
CREATE INDEX idx_crm_interaction_type ON crm_customer_interaction(interaction_type);
CREATE INDEX idx_crm_interaction_created ON crm_customer_interaction(created_at);
```

#### 5. 客户关注表 (crm_customer_follow)
```sql
CREATE TABLE crm_customer_follow (
    id          BIGSERIAL PRIMARY KEY,
    customer_id BIGINT NOT NULL REFERENCES crm_customer_info(id),
    user_id     BIGINT NOT NULL REFERENCES sys_user(id),
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(customer_id, user_id)
);
```

#### 6. 客户团队表 (crm_customer_team)
```sql
CREATE TABLE crm_customer_team (
    id              BIGSERIAL PRIMARY KEY,
    customer_id     BIGINT NOT NULL REFERENCES crm_customer_info(id),
    user_id         BIGINT NOT NULL REFERENCES sys_user(id),
    role            VARCHAR(50), -- owner, member, viewer
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(customer_id, user_id)
);
```

## 🔌 API设计

```http
GET /api/v1/customers
Query Parameters:
  - page: int (default: 1)
  - pageSize: int (default: 20)
  - keyword: string (客户名称搜索)
  - customerType: string (企业/个人)
  - level: string (A/B/C/D)
```

## 🤖 AI增强功能（与AI模块协作）

### 1. 智能信息补全
**触发时机：** 客户创建/编辑时
**功能：**
- 根据客户名称/网址自动补全企业信息
- 自动获取行业、规模、地址等信息
- 自动生成企业简介

### 2. 客户画像生成
**触发时机：** 信息补全后自动触发
**功能：**
- 分析企业业务领域
- 评估采购潜力
- 识别决策特点
- 生成标签建议

### 3. 联系人信息补全
**功能：**
- 根据邮箱/手机号判断有效性
- 根据职位判断决策力
- 推荐沟通策略
