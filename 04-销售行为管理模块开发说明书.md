# 模块四：销售行为管理模块开发说明书

## 📋 模块概述

### 模块名称
销售行为管理模块（Sales Activity Management）

### 模块定位
管理销售过程中的各类行为活动，包括客户拜访、跟进记录、任务管理等，帮助追踪销售过程和提升销售效率。

### 开发优先级
**P1（高优先级）** - 依赖客户、商机模块

---

## 🎯 模块目标

### 主要目标
1. 实现拜访记录管理（外勤拜访）
2. 实现跟进记录管理（电话、邮件、微信等）
3. 实现任务管理（待办事项、提醒）
4. 实现活动日历视图
5. 实现销售行为统计分析

### 业务价值
- 完整记录销售过程
- 规范销售行为
- 提高跟进效率
- 为销售分析提供数据基础

## 📐 数据库设计

### 表结构设计

#### 1. 拜访记录表 (crm_visit_record)
```sql
CREATE TABLE crm_visit_record (
    id                  BIGSERIAL PRIMARY KEY,
    visit_no            VARCHAR(50) NOT NULL UNIQUE,   -- 拜访编号
    visit_type          VARCHAR(50) NOT NULL,          -- 拜访类型: onsite/online

    -- 关联信息
    customer_id         BIGINT NOT NULL REFERENCES crm_customer_info(id),
    customer_name       VARCHAR(200),
    opportunity_id      BIGINT REFERENCES crm_opportunity_info(id), -- 关联商机

    -- 拜访对象
    contact_ids         BIGINT[],                      -- 参与联系人ID列表
    participant_names   VARCHAR(500),                  -- 参与人姓名（冗余）

    -- 时间地点
    visit_date          DATE NOT NULL,
    start_time          TIME,
    end_time            TIME,
    duration            INT,                           -- 时长(分钟)

    -- 地址信息（外勤拜访）
    province            VARCHAR(50),
    city                VARCHAR(50),
    address             VARCHAR(500),
    latitude            DECIMAL(10,7),                 -- 纬度
    longitude           DECIMAL(10,7),                 -- 经度

    -- 拜访内容
    purpose             VARCHAR(200),                  -- 拜访目的
    content             TEXT,                          -- 拜访内容/纪要
    customer_feedback   TEXT,                          -- 客户反馈
    next_plan           TEXT,                          -- 下一步计划

    -- 结果评估
    result_type         VARCHAR(50),                   -- 结果类型: positive/neutral/negative
    interest_level      SMALLINT,                      -- 兴趣等级 1-5
    purchase_intent     SMALLINT,                      -- 采购意向 1-5

    -- 附件
    photos              VARCHAR(500)[],                -- 照片URL
    attachments         VARCHAR(500)[],                -- 附件URL

    -- 费用
    expense_amount      DECIMAL(10,2),                 -- 拜访费用
    expense_desc        VARCHAR(200),                  -- 费用说明

    -- AI增强
    ai_summary          TEXT,                          -- AI生成的总结
    ai_action_items     JSONB,                         -- AI提取的行动项

    -- 元数据
    created_by          BIGINT REFERENCES sys_user(id),
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_crm_visit_customer ON crm_visit_record(customer_id);
CREATE INDEX idx_crm_visit_opportunity ON crm_visit_record(opportunity_id);
CREATE INDEX idx_crm_visit_date ON crm_visit_record(visit_date);
CREATE INDEX idx_crm_visit_created_by ON crm_visit_record(created_by);
```

#### 2. 跟进记录表 (crm_follow_record)
```sql
CREATE TABLE crm_follow_record (
    id                  BIGSERIAL PRIMARY KEY,
    follow_no           VARCHAR(50) NOT NULL UNIQUE,

    -- 关联信息
    customer_id         BIGINT NOT NULL REFERENCES crm_customer_info(id),
    customer_name       VARCHAR(200),
    opportunity_id      BIGINT REFERENCES crm_opportunity_info(id),
    contact_id          BIGINT REFERENCES crm_contact_info(id),

    -- 跟进类型
    follow_type         VARCHAR(50) NOT NULL,          -- call/email/wechat/message/other
    follow_direction    VARCHAR(20),                   -- inbound/outbound

    -- 跟进内容
    subject             VARCHAR(200),                  -- 主题
    content             TEXT,                          -- 内容

    -- 通话信息（电话类型）
    call_duration       INT,                           -- 通话时长(秒)
    call_recording_url  VARCHAR(500),                  -- 录音URL

    -- 邮件信息（邮件类型）
    email_from          VARCHAR(200),
    email_to            VARCHAR(500)[],
    email_cc            VARCHAR(500)[],
    email_subject       VARCHAR(500),
    email_body          TEXT,

    -- 结果
    response            TEXT,                          -- 对方回应
    result              VARCHAR(50),                   -- 结果评估

    -- 下次跟进
    next_follow_date    DATE,
    next_follow_note    TEXT,

    -- 附件
    attachments         VARCHAR(500)[],

    -- AI增强
    ai_summary          TEXT,                          -- AI总结
    ai_sentiment        VARCHAR(20),                   -- 情感分析: positive/neutral/negative

    -- 元数据
    created_by          BIGINT REFERENCES sys_user(id),
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_crm_follow_customer ON crm_follow_record(customer_id);
CREATE INDEX idx_crm_follow_opportunity ON crm_follow_record(opportunity_id);
CREATE INDEX idx_crm_follow_type ON crm_follow_record(follow_type);
CREATE INDEX idx_crm_follow_created_by ON crm_follow_record(created_by);
CREATE INDEX idx_crm_follow_created_at ON crm_follow_record(created_at);
```

#### 3. 任务管理表 (crm_task_info)
```sql
CREATE TABLE crm_task_info (
    id                  BIGSERIAL PRIMARY KEY,
    task_no             VARCHAR(50) NOT NULL UNIQUE,
    task_title          VARCHAR(200) NOT NULL,

    -- 关联信息
    customer_id         BIGINT REFERENCES crm_customer_info(id),
    opportunity_id      BIGINT REFERENCES crm_opportunity_info(id),
    related_object_type VARCHAR(50),                   -- customer/opportunity/contact
    related_object_id   BIGINT,

    -- 任务类型
    task_type           VARCHAR(50) NOT NULL,          -- call/visit/meeting/followup/other
    task_category       VARCHAR(50),                   -- 任务分类

    -- 时间信息
    due_date            DATE NOT NULL,                 -- 截止日期
    due_time            TIME,                          -- 截止时间
    reminder_time       TIMESTAMP,                     -- 提醒时间
    duration            INT,                           -- 预计时长(分钟)

    -- 优先级与状态
    priority            VARCHAR(20) DEFAULT 'medium',  -- high/medium/low
    status              VARCHAR(20) DEFAULT 'pending', -- pending/in_progress/completed/cancelled

    -- 任务详情
    description         TEXT,
    location            VARCHAR(200),                  -- 地点
    notes               TEXT,

    -- 完成信息
    completed_at        TIMESTAMP,
    completion_note     TEXT,

    -- 重复设置
    is_recurring        BOOLEAN DEFAULT FALSE,
    recurrence_rule     JSONB,                         -- 重复规则
    parent_task_id      BIGINT REFERENCES crm_task_info(id),

    -- 参与人
    assigned_to         BIGINT REFERENCES sys_user(id),
    created_by          BIGINT REFERENCES sys_user(id),

    -- 附件
    attachments         VARCHAR(500)[],

    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_crm_task_assigned ON crm_task_info(assigned_to);
CREATE INDEX idx_crm_task_status ON crm_task_info(status);
CREATE INDEX idx_crm_task_due_date ON crm_task_info(due_date);
CREATE INDEX idx_crm_task_customer ON crm_task_info(customer_id);
CREATE INDEX idx_crm_task_opportunity ON crm_task_info(opportunity_id);
```

#### 4. 日程安排表 (crm_schedule)
```sql
CREATE TABLE crm_schedule (
    id                  BIGSERIAL PRIMARY KEY,
    schedule_title      VARCHAR(200) NOT NULL,

    -- 时间信息
    start_time          TIMESTAMP NOT NULL,
    end_time            TIMESTAMP NOT NULL,
    is_all_day          BOOLEAN DEFAULT FALSE,

    -- 关联信息
    customer_id         BIGINT REFERENCES crm_customer_info(id),
    opportunity_id      BIGINT REFERENCES crm_opportunity_info(id),
    task_id             BIGINT REFERENCES crm_task_info(id),

    -- 类型
    schedule_type       VARCHAR(50),                   -- meeting/call/training/other
    location            VARCHAR(200),

    -- 详情
    description         TEXT,
    attendees           JSONB,                         -- 参与人
    -- [{"userId": 1, "userName": "张三", "status": "accepted"}]

    -- 提醒
    reminder_minutes    INT[],                         -- 提醒时间点

    -- 重复
    is_recurring        BOOLEAN DEFAULT FALSE,
    recurrence_rule     JSONB,

    -- 状态
    status              VARCHAR(20) DEFAULT 'scheduled', -- scheduled/completed/cancelled

    created_by          BIGINT REFERENCES sys_user(id),
    created_at          TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_crm_schedule_start_time ON crm_schedule(start_time);
CREATE INDEX idx_crm_schedule_created_by ON crm_schedule(created_by);
```

#### 5. 任务提醒日志表 (crm_task_reminder_log)
```sql
CREATE TABLE crm_task_reminder_log (
    id              BIGSERIAL PRIMARY KEY,

    user_id         BIGINT NOT NULL REFERENCES sys_user(id),
    reminder_time   TIMESTAMP NOT NULL,
    reminder_type   VARCHAR(50),     -- email/sms/push
    status          VARCHAR(20),     -- sent/failed

    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
```
