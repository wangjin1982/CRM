# 模块五：AI智能增强模块开发说明书

## 📋 模块概述

### 模块名称
AI智能增强模块（AI Intelligence Enhancement）

### 模块定位
系统的核心差异化模块，通过集成大模型API（GLM等）实现智能信息补全、数据分析和风险预警等功能。

### 开发优先级
**P1（高优先级）** - 可与业务模块并行开发，通过API解耦

---

## 🎯 模块目标

### 主要目标
1. 实现客户/商机信息的智能补全
2. 实现销售数据的智能分析
3. 实现商机风险的智能预警
4. 提供自然语言查询接口
5. 实现智能推荐与建议

### 业务价值
- 减少数据录入工作量
- 提升数据分析效率
- 提前识别销售风险
- 提供智能化决策支持

#### 3. AI请求日志表 (ai_request_log)
```sql
CREATE TABLE ai_request_log (
    id              BIGSERIAL PRIMARY KEY,
    request_id      VARCHAR(100) UNIQUE,   -- 请求ID
    request_type    VARCHAR(50) NOT NULL,   -- enrichment, analysis, summary, risk

    -- 请求信息
    model           VARCHAR(50),
    prompt_tokens   INT,
    system_prompt   TEXT,
    user_prompt     TEXT,

    -- 响应信息
    response        TEXT,
    response_tokens INT,
    total_tokens    INT,
    finish_reason   VARCHAR(50),

    -- 状态
    status          VARCHAR(20),           -- success, error, timeout
    error_msg       TEXT,

    -- 性能
    latency         INT,                   -- 响应时间(ms)

    -- 元数据
    related_object_type VARCHAR(50),       -- customer, opportunity
    related_object_id   BIGINT,
    created_by      BIGINT REFERENCES sys_user(id),
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_ai_request_type ON ai_request_log(request_type);
CREATE INDEX idx_ai_request_created ON ai_request_log(created_at);
CREATE INDEX idx_ai_request_object ON ai_request_log(related_object_type, related_object_id);
```

## 🤖 AI功能详细设计

### 1. 信息智能补全

#### 触发场景
| 场景 | 触发条件 | 补全内容 |
|-----|---------|---------|
| 新建客户 | 输入客户名称后 | 企业基本信息、行业、规模 |
| 客户详情 | 点击"AI补全"按钮 | 完善企业简介、标签 |
| 新建联系人 | 输入邮箱/手机 | 验证有效性、职位判断 |
| 商机创建 | 关联客户后 | 根据客户历史生成机会评估 |

#### 补全流程
```
用户输入 → 数据提取 → LLM请求 → 结果解析 → 数据存储 → 前端展示
```

### 2. 风险智能预警

#### 预警规则
| 预警类型 | 触发条件 | 风险等级 |
|---------|---------|---------|
| 商机停滞 | 阶段停留 > 建议天数*1.5 | 中 |
| 商机严重停滞 | 阶段停留 > 建议天数*2 | 高 |
| 活动减少 | 30天无活动记录 | 中 |
| 长期无活动 | 60天无活动记录 | 高 |
| 预计成交逾期 | 超过预计成交日期30天 | 中 |
| 大额商机风险 | 金额>100万且停留时间长 | 高 |

#### AI增强预警
- 分析历史类似商机的成交率
- 评估当前活动的有效性
- 识别隐性的风险因素

### 3. 销售数据分析

#### 分析维度
- 漏斗转化分析
- 成交周期分析
- 客户行为模式
- 销售活动效率
- 业绩趋势预测

#### 分析输出
```
{
    "overview": "整体情况概述",
    "charts": ["图表配置数据"],
    "insights": ["洞察结论"],
    "recommendations": ["改进建议"]
}
```
